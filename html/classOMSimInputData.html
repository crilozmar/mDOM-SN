G4Material * OMSimInputData::GetMaterial ( G4String  pName)

Get a G4Material. In order to get custom built materials, method SearchFolders() should have already been called. Standard materials from Geant4 are also transfered directly (if found through FindOrBuildMaterial()). Materials defined by arguments of the main should start with "arg" and then we get the material name from the arrays depending on the global integers (gGel, gGlass, gEnvironment..) .

Parameters
pNamename of the (custom) material or "argVesselGlass", "argGel", "argWorld" for argument materials
Returns
G4Material

void OMSimInputData::MaterialFromFile ( G4String  pFilename)
private

Defines new material from data in json-file. The prefix of the filename tells this function what kind of material we are handling: "RiAbs" : Material with refraction index and absorption length. "Ri" : Material with refraction index only "NoOptic": Material has no optics defined "ICE" : This is a special file with data from Icecube. Here we call several function for calculating the optical properties

Parameters
pFilenamename of the file with material properties
